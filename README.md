# Fénix Viral – Landing Profesional

Este repositorio contiene una única página **index.html** totalmente autocontenida que incluye:

- **Hero** con gradiente indigo‑naranja, título con sombra y CTA de llamada.
- **Navbar** fija, semitransparente y responsive (hamburguesa en móvil).
- **Tarjetas de servicios** con iconos emoji, hover elevation y tipografía personalizada.
- **Sección de estadísticas** con contadores animados en JavaScript.
- **Imagen real** (`static/img/imagen_1.jpg`) usada en la sección “Sobre Nosotros”.
- **Botón flotante** “Ver Carta Digital” que abre un modal con un QR code generado vía API.
- **Botón WhatsApp** flotante (verde) que enlaza a `https://wa.me/123456789`.
- **Footer** completo con logo, enlaces rápidos y datos de contacto.
- **Animaciones** de fade‑in al hacer scroll y transiciones suaves en botones y tarjetas.
- **Responsive design** mobile‑first, sin scroll horizontal.

Todo el CSS está dentro de `<style>` y el JavaScript dentro de `<script>`; no se usan frameworks ni procesos de build. La página supera las 300 líneas de código para garantizar una experiencia rica y completa.
